# Osyntra

Zestaw narzędzi do osint i analizy metadanych.

## Dostępne narzędzia

- 🧠 **MindDork Tool** - wizualna mapa dorków, linków i notatek do białego wywiadu
- 🔍 **Analizator plików i metadanych** - podgląd plików, analiza EXIF i wyciąganie informacji z metadanych

## Instalacja

```bash
npm install
```

## Uruchomienie

```bash
npm run dev
```

## Budowanie

```bash
npm run build
```

## Użycie

1. Zaloguj się lub zarejestruj (dane zapisywane są lokalnie)
2. Wybierz narzędzie z menu
3. Skorzystaj z dostępnych funkcji zgodnie z ich opisem

## Ważne informacje

- Narzędzie służy wyłącznie do celów legalnych i etycznych
- Używaj zgodnie z prawem i regulaminami usług
